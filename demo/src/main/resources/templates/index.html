¿Cómo funciona SAV12?

SAV12 es una plataforma para registrar, priorizar y dar seguimiento a incidencias técnicas dentro de una organización.

Flujo básico

  1. Registro de incidencia: el usuario describe el problema.
  2. Clasificación: se asigna prioridad y categoría.
  3. Asignación: el sistema asigna la incidencia a un técnico.
  4. Seguimiento: se registran avances y cambios de estado.
  5. Cierre: se documenta la solución y se mide el tiempo (SLA).

Roles principales

  • Usuario: reporta incidencias y consulta el estado.
  • Técnico: atiende y actualiza los reportes.
  • Administrador: configura prioridades, SLAs y usuarios.

Esta es una versión de demostración de la plataforma SAV12. Próximamente se integrarán formularios reales para levantar reportes.
